# 🎮 Tic-Tac-Toe Game (Python)

A fun and interactive **Tic-Tac-Toe game** built using Python where the user can play against the computer.  
This project includes multiple features like difficulty levels, score tracking, and replay functionality.

- ✅ Input validation  
- 🎯 Win / Draw detection  

---

## 🧠 Game Logic

- The board is represented using a list of 9 positions  
- Player uses **X** and Computer uses **O**  
- The computer:
  - Makes random moves in Easy mode  
  - Tries to win or block the player in Hard mode
